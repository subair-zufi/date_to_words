

`date_to_words` converts `DateTime` to English words



## Usage


```dart
final DateTime _myDate = DateTime.now();
final _dateInWord= DateToWord.instance.convert(_myDate);
print(_dateInWord);
// Result sample: first april two thousand twenty two
```
